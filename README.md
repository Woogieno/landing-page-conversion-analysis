# 랜딩페이지 도입을 통한 마케팅 전환율 개선 분석

> **분석 기간:** 2025년 12월 (도입 전) vs 2026년 2월 (도입 후)  
> **사용 도구:** Google Analytics 4 (GA4), 사내 문의 데이터 , Google SpreadSheet

> **역할:** 마케팅팀 팀장 — 문제 정의, 가설 설정, 랜딩페이지 기획·제작, 성과 검증

---

## 1. 배경 및 문제 정의

마케팅 광고를 집행 중임에도 실제 문의 전환이 기대치에 미치지 못하는 상황이 지속됨.  
GA4 방문 페이지 데이터와 사내 문의 데이터를 교차 분석하여 원인을 파악하고자 함.

> **기존 홈페이지(/) 구조의 문제:**  
> 홈페이지는 기업 소개, MOU 체결, 솔루션 상세, 수상 경력 등 **기업 홍보 목적**으로 구성되어 있음.  
> 반면 광고를 통해 유입되는 방문자는 **즉각적인 피해 해결**이 필요한 상태로, 페이지 목적과 방문자 니즈 간 불일치가 발생.

### 도입 전 GA4 데이터 (2025년 12월)

| 페이지 | 세션수 | 비율 | 세션 이벤트 전환율 |
|---|---|---|---|
| / (홈) | 6,464 | 83.7% | 5.21% |
| /board/board_list | 322 | 4.17% | 10.25% |
| /board/board_view | 163 | 2.11% | 4.29% |
| /board/board_write | 84 | 1.09% | 51.19% |
| **합계** | **7,723** | **100%** | **7.81%** |

### 핵심 문제 발견

- 전체 세션의 **83.7%** 가 홈(/)으로 유입되지만, 이벤트 전환율은 **5.21%** 에 불과
- 문의 작성 페이지(/board/board_write)에 도달한 세션은 전체의 **1.09%** 에 그침
- 즉, **대다수의 방문자가 홈에서 이탈하며 문의로 이어지지 않음**

## 2. 가설 설정

> **"홈페이지 개편보다, 고객 니즈에 직접 답하는 전용 랜딩페이지를 제작하면  
> 문의 전환율을 더 효과적으로 높일 수 있을 것이다."**

### 가설의 근거

- 홈페이지는 브랜드 전체를 소개하는 구조 → 방문자가 원하는 정보 탐색에 마찰 발생
- 실제 고객 문의 패턴 분석 결과, 반복적으로 묻는 질문 유형이 존재
- 광고 트래픽은 특정 니즈를 가진 사용자 → 맞춤 콘텐츠 제공 시 전환율 향상 예상

### 랜딩페이지 설계 방향

1. 고객이 자주 묻는 질문(FAQ) 중심 콘텐츠 구성
2. 스크롤 흐름에 따라 자연스럽게 문의로 유도하는 CTA 배치
3. 불필요한 탐색 없이 핵심 정보만 전달하는 단일 페이지 구조

---

## 3. 실행

- 랜딩페이지(`/ads`) 직접 기획 및 제작 (Vibe Coding 활용)
- 광고 캠페인의 랜딩 URL을 홈(/)에서 `/ads`로 전환
- 배포 후 GA4로 유입·전환 지표 지속 모니터링

## 5. 해석 및 인사이트

### 전환율이 소폭 하락한 이유

전체 세션 이벤트 전환율이 7.81% → 6.48%로 하락했지만, 이를 단순 부정적 결과로 해석하면 오류임.

- 도입 후 신규 사용자 비율이 **86.84%** 로, 브랜드를 처음 접하는 콜드 트래픽이 대폭 유입됨
- 콜드 트래픽은 구조적으로 전환율이 낮기 때문에, 전체 평균 전환율의 분모가 커진 것
- 실제로 절대 이벤트 수(+37.4%)와 사내 문의·매출 지표는 명확히 개선됨
- **랜딩페이지(/ads) 자체 전환율 6.35%는 기존 홈(/) 전환율 5.21%보다 높음**

### 결론

랜딩페이지 도입은 단순 전환율 개선을 넘어, **광고 유입 트래픽의 규모 자체를 확대**하면서  
절대적인 문의 수와 매출을 동시에 끌어올리는 데 기여함.  
데이터 기반 가설 수립 → 실행 → 검증의 사이클이 실제 비즈니스 성과로 이어진 사례.

## 6. 한계 및 개선 방향

### 서비스 구조적 특성 (분석 전제 조건)

해당 서비스는 **몸캠피싱 피해를 기술적으로 해결하는 1회성 서비스**로,  
문제가 해결된 고객은 재방문할 이유가 없는 구조임.  
따라서 아래 수치들은 서비스 실패가 아닌 **정상적인 비즈니스 특성**으로 해석해야 함:

- 신규 사용자 비율 86.84% → 재방문이 없는 구조에서 당연한 결과
- 리텐션 지표 부재 → 측정 대상 자체가 아님
- 핵심 성과 지표는 **신규 유입 → 최초 문의 전환**에 집중해야 함

### 분석 한계

- 도입 전후 기간(12월 vs 2월) 차이로 인한 **계절성 변수** 통제 필요
- 광고 예산 변화 여부에 따른 트래픽 증가분 구분 분석 필요
- `/ads` 페이지 내 **스크롤 깊이, 클릭 히트맵** 추가 분석으로 CTA 최적화 여지 있음
- A/B 테스트 설계를 통해 랜딩페이지 요소별 기여도 측정 필요
